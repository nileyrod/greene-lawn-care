# Greene's Lawn Care Website

## Overview
A modern, responsive single-page marketing website for Greene's Lawn Care, a local lawn care company serving Richmond, VA and surrounding areas. The site is built with pure HTML, CSS, and vanilla JavaScript - no frameworks required.

## Business Information
- **Business Name**: Greene's Lawn Care
- **Location**: Richmond, VA
- **Phone**: (540) 424-1690
- **Email**: cgreeneslawncare@gmail.com
- **Service Area**: Richmond and nearby areas

## Project Structure
- **index.html**: Complete single-file website with embedded CSS and JavaScript

## Website Sections
1. **Navigation Bar**: Sticky header with logo and smooth-scroll menu links, responsive hamburger menu for mobile
2. **Hero Section**: Eye-catching intro with call-to-action buttons for phone and quote requests
3. **Services Section**: Grid of 6 lawn care services with descriptions
4. **Gallery Section**: "Our Past Work" showcase with real project photos and hover zoom effects
5. **Reviews Section**: Customer testimonials with 5-star ratings
6. **About Section**: Company values and why to choose Greene's Lawn Care
7. **Contact Form**: Quote request form with client-side validation
8. **Footer**: Business info, contact details, and social media links

## Features
- Mobile-first responsive design
- Smooth scrolling navigation
- Interactive hamburger menu for mobile devices
- Form validation with success messages
- Back-to-top button that appears on scroll
- Green and white color scheme with professional styling
- Accessible with semantic HTML5 elements
- Google Fonts (Poppins)

## Technology Stack
- HTML5 (semantic markup)
- CSS3 (embedded, mobile-first, flexbox and grid layouts)
- Vanilla JavaScript (form validation, smooth scrolling, mobile menu)

## How to Run
The website is served using Python's built-in HTTP server on port 5000:
```
python -m http.server 5000
```
